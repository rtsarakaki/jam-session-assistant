# Jam Session Assistant - Product Architecture

Esta pasta concentra a documentacao de produto sob a perspectiva de arquitetura de negocio.

## Estrutura

- `01-business-context.md` - contexto, atores e fronteiras do produto.
- `02-capability-map.md` - mapa de capacidades de negocio.
- `03-value-streams.md` - fluxos de valor ponta a ponta.
- `04-domain-and-information.md` - modelo de dominio e informacao.
- `05-operating-model-and-kpis.md` - modelo operacional, metricas e governanca.

## Como usar

1. Ler em sequencia para entendimento completo.
2. Usar os diagramas em reunioes de alinhamento (produto, engenharia e operacoes).
3. Atualizar os documentos quando houver mudancas relevantes de estrategia.
